# Creating Projects in Jot

Turn your ideas into organized, actionable projects.

---

## Quick Start

### 1. Create a New Project

Tap the **+** button in the Projects tab and fill in:

- **Name** - What's this project about?
- **Description** - Why does it matter?
- **Status** - Planning, Active, Paused, or Completed
- **Due Date** - Optional deadline
- **Tags** - For easy filtering

### 2. Build Your Project

Your project includes a README where you can:

- Write an overview
- List goals and objectives
- Track tasks with checkboxes
- Add notes and resources
- Attach files

### 3. Work the Plan

- Check off tasks as you complete them
- Update your README as things change
- Review progress regularly

## Structuring Your Project

### Basic Template

```markdown
# Project Name

## Overview

Brief description of what this is about.

## Goals

- Primary goal
- Secondary goal

## Tasks

### This Week

- [ ] Task 1 due:2024-12-15
- [ ] Task 2 @person
- [ ] Task 3 prio:high

### Next Week

- [ ] Task 4
- [ ] Task 5

## Notes

Key information and context.

## Next Actions

1. First step
2. Then this
3. Finally that
```

---

## Task Syntax

Make tasks more useful with special syntax:

**Due Dates**

```markdown
- [ ] Task name due:2024-12-15
```

**Assignments**

```markdown
- [ ] Task name @person
- [ ] Another task @team-member
```

**Priority**

```markdown
- [ ] Urgent task prio:high
- [ ] Important task prio:medium
- [ ] Can wait prio:low
```

**Combine Them**

```markdown
- [ ] Launch homepage @designer due:2024-12-20 prio:high
```

## Advanced: Import Projects

Already wrote your project in another app? Import it!

### From Markdown Files

1. Go to **Settings ‚Üí Data Management ‚Üí Import**
2. Select your `.md` file
3. Jot imports it as a note
4. Create a project and link the note

### From Other Apps

- **Notion/Obsidian**: Export as markdown
- **Emails**: Copy/paste into new note
- **Documents**: Convert to markdown first

### Pro Tip: Templates

Create a project template once, export it, and reuse it for similar projects!

## FAQ

**Can I have multiple projects?**
Yes! Create as many as you need. Use tags to organize them.

**Can others collaborate?**
Currently Jot is single-user. You can share by exporting and sending the markdown file.

**What if I don't know all the tasks yet?**
That's fine! Start with big phases, add tasks as you go.

**Can I track time or budgets?**
Add those as custom notes sections for now. Dedicated tracking coming later.

**Can I duplicate a project?**
Export as markdown, save as template, import to create new similar projects.
