# Keyboard Shortcuts

## Global
- **Preferences**: ⌘,
- **New Note**: ⌘N
- **Search**: ⌘F
- **Quit**: ⌘Q
- **Show Shortcuts**: ⌘/

## Editing (Common)
- **Undo / Redo**: ⌘Z / ⇧⌘Z
- **Cut / Copy / Paste**: ⌘X / ⌘C / ⌘V
- **Select All**: ⌘A

## Notes
- Jot saves automatically.
- **Open Selected/Open Note in New Window (Desktop)**: ⌘⇧Enter (macOS), Ctrl+Shift+Enter (Windows)

## Tip
If you’re not sure what works in the current screen, press **⌘/** to see the shortcut overlay.
